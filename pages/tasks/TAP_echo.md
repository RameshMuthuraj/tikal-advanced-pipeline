# TAP_echo

***Echo text with time-stamp***

# Task usage

TAP_echo(arguments)

Arguments:

| Argument name and type | Description | Default Value |
| ------------- | ----------- | ------------- |
| String text| Text to display| N/A|

# Examples

### Usage
TAP_echo ("Hello TAP!")

### Output
[20170715-05:40:11.393] Hello TAP!
